**AVOXX is in the alpha stage. Please do not use this package in production environments. The first stable release will be published ~ 2018.**

## Requirements
* [PSR-7 HTTP message interfaces](https://github.com/container-interop/fig-standards/blob/master/proposed/http-message.md)

## Installation

The AVOXX packages utilizes [Composer](https://getcomposer.org) to manage their dependencies.
So, before using any of the AVOXX packages, make sure you have Composer installed on your machine.

    composer require avoxx/psr7 dev-master

This will install the `avoxx/psr7` package and all required dependencies.
